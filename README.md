# FitZone Gym Website

## About This Project

This is a responsive business website developed for FitZone Gym. The website consists of four pages:

- Home
- About
- Services
- Contact

### Main Features

- Responsive Design
- Navigation Bar
- Hero Section
- About Section
- Services Section
- Contact Form
- Form Validation using JavaScript
- CSS Grid & Flexbox Layout
- Mobile Responsive Design
